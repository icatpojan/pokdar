# Menjalankan Aplikasi 'sistem'

Ikuti langkah-langkah berikut untuk menjalankan aplikasi secara lokal menggunakan server pengembangan bawaan PHP:

1. Buka terminal di direktori proyek:
   ```bash
   cd /Users/mac/codingan/sistem
   ```

2. Jalankan perintah server bawaan PHP:
   // turbo
   ```bash
   php -S localhost:8000
   ```

3. Akses aplikasi melalui browser di alamat:
   [http://localhost:8000](http://localhost:8000)

4. Untuk berhenti, tekan `Ctrl + C` di terminal.
